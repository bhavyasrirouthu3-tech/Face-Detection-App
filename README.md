# Face Detection App using OpenCV

## Project Overview

This project detects human faces in real time using OpenCV and a webcam.

## Features

- Real-time face detection
- Webcam support
- Haar Cascade Classifier
- Green rectangle around detected faces
- Press Q to exit

## Technologies Used

- Python 3.13
- OpenCV
- NumPy

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
py -3.13 main.py
```
